13.Выгрузка системы (System Shutdown).

   Разработать программу, которая позволяет выполнять одно из следующих действий: завершение текущего сеанса, перезагрузку компьютера, выключение компьютера, переход в спящий режим.

Можно предложить следующий вариант реализации.

   При запуске программы открывается окно с тремя кнопками, как представлено на рисунке (Рис 13.1).


Рис 13.1 Внешний вид приложения

   При нажатии на кнопку Logoff производится завершение всех текущих процессов и завершение сеанса текущего пользователя.
   Кнопка Shutdown служит для завершения работы Windows и выключения питания компьютера.
   При нажатии на кнопку Reboot производится перезагрузка системы.
   Нажатие кнопки Cancel закрывает окно и завершает работу программы. Аналогичные действия производятся при нажатии на клавишу ESC.
