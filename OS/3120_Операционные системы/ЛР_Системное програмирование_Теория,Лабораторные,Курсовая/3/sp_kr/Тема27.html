27.Сжатие данных (Data Decompression Library).

   

Функции, используемые Библиотекой восстановления сжатых данных

Нижеследующая таблица описывает каждую функцию, находящуюся в LzExpand.dll. Они используются, чтобы развернуть файлы, которые сжимались по алгоритму Лемпеля - Зива.

Функция Предназначение
GetExpandedName Извлекает первоначальное имя сжатого файла, если в ходе  сжатия файла использовался параметр /r.
LZClose Закрывает файл, который был открыт когда приложение вызвало функцию LZOpenFile.
LZCopy Копирует исходный файл в выходной файл. Если исходный файл сжат, эта функция создает распакованный выходной файл. Если исходный файл не сжат, эта функция делает копию первоначального файла. Эта функция предназначена для операций копирования отдельно взятого файла.
LZInit Назначает память для структур используемых, чтобы разворачивать файлы, а затем создает и инициализирует их.
LZOpenFile Создает, открывает, повторно открывает или удаляет указанный файл и возвращает дескриптор, который идентифицирует его.
LZRead Читает максимально заданное число байтов из файла и копирует их в буфер. Если файл сжат, эта функция разворачивает байты перед копированием их в принимающий буфер.
LZSeek Помещает указатель позиции в файле внутри распакованного образа сжатого файла. Приложение вызывает эту функцию для того, чтобы поместить указатель перед вызовом функции LZRead.
