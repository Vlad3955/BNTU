Debug – программа разработки и отладки программ на языке Ассемблер x86

 

 

Программа Debug – назначение, способ применения, загрузка

 

Команды программы Debug:

 

A(Assemble)                 Команда трансляции

С(Compare)                  Команда сравнения

D(Dump)                      Команда вывода на экран содержимого памяти

E(Enter)                        Команда ввода

F (FILL)                        Команда заполнения памяти

G(Go)                            Команда выполнения программы

H(Hexarithmetic)          Команда шестнадцатиричной арифметики

M(Move)                       Команда передачи содержимого памяти

P(Proceed)                     Команда возобновления

Q (Quit)                        Завершение программы DEBUG

R(Register)                    Команда операции с регистрами

S(Search)                      Команда поиска.

Т(Тгасе)                        Команда отслеживания

U(Unassemble)             Команда дизассембляции
