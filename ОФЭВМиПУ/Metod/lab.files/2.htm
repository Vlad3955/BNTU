Лабораторная работа №2

 «Изучение команд пересылки данных»

 

Цель работы: изучить способы адресации данных микропроцессором x86 и применение этих методов при пересылке данных.

 

Оборудование: IBM - совместимый персональный компьютер, работающий под управлением операционной системы, совместимой с MS-DOS®.

 

Программное обеспечение: Программа DEBUG

 

Краткие теоретические сведения.

 

1.   Методы адресации данных

2.   Команды пересылки данных

 

 

Индивидуальное задание.

 

Выбор индивидуального варианта производить по следующей методике:

- в зачетке каждого студента имеется индивидуальный порядковый номер. Он расположен после левой наклонной черты. Например, номер зачетки студента 417 211/15. Следовательно, его индивидуальный номер – 15;

- в первой таблице студент выбирает задание с номером, который равен сумме цифр 1 и 5, т.е. вариант номер 6;

- во второй таблице задание с номером 5;

 

1. Объявить массив заданного размера и с заданным количеством элементов (табл. 1).

2. Проинициализировать массив 0, используя произвольный метод адресации.

3. В каждый элемент массива переслать число (табл.1), используя метод адресации согласно варианту (табл.2).
